## Usage Notes

All models are free for commercial use in the iRis AR platform. The CC BY 4.0 licensed models require attribution, which should be displayed in the application where the models are used.

For production use with branded items, please ensure you have appropriate licenses for any trademarked designs.

## Optimization Details

All models have been optimized for mobile AR performance:
- Mesh decimation where appropriate
- Texture atlas generation
- Draco/Meshopt compression
- Normalized scale (meters)
- Centered pivot points
